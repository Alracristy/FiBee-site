FiBeeBem-vindos ao FiBee, um projeto desenvolvido por alunos do Instituto federal do 3 ano B informática (2021/2023-2024) na matéria Projeto e Desenvolvimento de Sistemas, integrada a matéria de Desenvolvimento Web

PLANEJAMENTOS ESTRATÉGICOS DA EMPRESA

  • Chat para conversa

    Futuramente, nossa empresa pretende criar um sistema de chat para o usuário tirar duvidas com o professor ou com outro usuário.

  • Tempo de login na plataforma

    Com base na nossa empresa, pretendemos iniciar nossa plataforma de quiz com 30 horas de login incluindo questões de logica de programação nível fácil, médio e difícil.

  • Fácil download

    Inicialmente compartilharemos um link para download da nossa plataforma, mas futuramente iremos atribuí-lo no google Play Store e Apple Store.

QUER TESTAR NOSSO APLICATIVO?

Com base na nossa plataforma que é compativel para qualquer dispositivo movel, ele possui uma leitura dinâmica fazendo com que o usuário não fique intediado com a leitura das questões.

Apresentação Geral

O presente trabalho teve como finalidade a contextualização dos obstáculos no aprendizado de lógica de programação ao ser introduzida nos cursos de Computação. A falta de instrução no ensino do pensamento lógico instigou o erro logo no início dos estudos o que se tornou um obstáculo difícil de se corrigir.

Por conta de dados gerados através de entrevistas com docentes, foi possível uma melhor visão de como a prática e o uso do raciocínio lógico oferecidos pelo aplicativo FiBee apresentaram a obtenção de soluções para um aprendizado eficaz, tendo como finalidade a resolução dos problemas propostos.


O aplicativo FiBee então tem como resultado não a proposta de um aprendizado de lógica de programação em pouco tempo, mas sim a prática do raciocínio lógico para uma melhor compreensão pessoal do estudante.
